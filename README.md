# 🎮 Tris Svanente

**[Gioca subito →](https://imperostefano.github.io/TicTacToe/)**

Tris con una svolta: **massimo 4 simboli** per giocatore. Il quinto cancella il più vecchio.

**Fatto da Stefano Kolta**

---

## 🎯 Come si gioca

1. Scegli **locale** (stesso schermo) o **online** (condividi un codice breve, es. `tris123`).
2. Inserisci i nickname.
3. Piazza i simboli a turno. Se hai già 4 pezzi, il più vecchio **svanisce** (e lampeggia prima).
4. Vince chi fa **tre in fila**. Dopo 16 mosse totali → pareggio.
5. Punteggio accumulato round dopo round.
